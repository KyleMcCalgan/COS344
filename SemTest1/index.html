In Scope

Chapter 1

Introduction to Computer Graphics

Applications of CG, rendering paradigms (object-order vs image-order), graphics APIs, and the basic graphics pipeline concept.

rendering pipeline OpenGL
Chapter 3

Raster Images

Pixels, image geometry, RGB color model, bit depth, gamma correction, alpha compositing, and image file formats.

pixels RGB alpha gamma
Chapter 4

Ray Tracing

Basic ray tracing algorithm, perspective vs parallel projection, computing viewing rays, ray-sphere and ray-triangle intersection.

viewing rays intersection projection
Chapters 4.5 + 5

Surface Shading

Light sources (point, directional, ambient), Lambertian diffuse reflection, Blinn-Phong specular model, BRDF, and irradiance equations.

Lambertian Blinn-Phong BRDF irradiance
Chapter 7

Transformations

2D & 3D linear transformations (scale, shear, reflection, rotation), homogeneous coordinates, translation, affine transforms, and inverse matrices.

matrices rotation homogeneous
Chapter 9

Rasterization

Barycentric coordinates, triangle rasterization, line drawing, z-buffer algorithm, Gouraud and Phong shading, painters algorithm.

barycentric z-buffer Gouraud
Chapter 17

Graphics Hardware & OpenGL

GPU architecture, heterogeneous multiprocessing, OpenGL state machine, VBOs, VAOs, GLSL vertex & fragment shaders, and primitive types.

GPU VBO VAO GLSL shaders
